# EKB Icon System v2 — «Soft Pastel Tiles»

v1 не трогаем. v2 = тот же monoline + squircle, но **у каждой категории свой мягкий пастельный фон**.

## Принципы (как у референса)
| Параметр | Значение |
|----------|----------|
| Контейнер | Squircle, сильное скругление |
| Стиль | Monoline outline, rounded caps/joins |
| Fill | Нет — только контур |
| Stroke weight | Один на всю систему |
| 3D / тень / emoji | Запрещено |
